FAQ

這套模板可以直接上線嗎？

可以，它是靜態站。你可先用 GitHub Pages / Cloudflare Pages 上線，再慢慢換外觀與內容。

我要怎麼把牌庫換成我的？

打開 data.js，把 UT.cards 的 name/oneLine/actionText/microStep/imageUrl 換成你的牌資料。
